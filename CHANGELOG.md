# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2025-01-12

### Added
- Initial project setup with Rust workspace
- Shared library for common functionality
- Web client using Yew framework
- Server using Axum framework
- WASM web package for browser integration
- Nix flake for reproducible builds
- GitHub Actions CI/CD pipeline
- Omnix integration for cross-platform builds
